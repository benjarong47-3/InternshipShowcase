## 🏫 เกี่ยวกับสถานที่ฝึกงาน
**โรงเรียนยุวฑูตศึกษาพัฒนาราชพฤกษ์ (Ambassador Bilingual Academy)** สถาบันการศึกษาหลักสูตรทวิภาษาที่มุ่งเน้นการสร้างสภาพแวดล้อมการเรียนรู้ที่ทันสมัย โดยแผนก IT มีบทบาทสำคัญในการขับเคลื่อนระบบการศึกษาผ่านเทคโนโลยีสารสนเทศ

## 🛠 บทบาทและหน้าที่รับผิดชอบ (Responsibilities)
ในระหว่างการฝึกงาน ผมได้รับมอบหมายให้ดูแลและสนับสนุนงานด้านต่าง ๆ ดังนี้:
* **IT Support & Troubleshooting:** แก้ไขปัญหาการใช้งาน Hardware และ Software ให้แก่บุคลากรและนักเรียน
* **Network Infrastructure:** เรียนรู้การจัดการระบบเครือข่ายภายในสถานศึกษา เพื่อให้การเรียนการสอนดำเนินไปอย่างราบรื่น
* **Digital Media Support:** ช่วยเหลือการผลิตสื่อและการใช้เครื่องมือดิจิทัลภายในโรงเรียน
* **System Maintenance:** ตรวจสอบและบำรุงรักษาอุปกรณ์คอมพิวเตอร์และระบบสารสนเทศส่วนกลาง

## 🙏 คำขอบคุณ (Acknowledgments)
